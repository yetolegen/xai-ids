# xai-ids

Explainable intrusion detection experiments focused on practical model behavior and error analysis.

The first dataset is NSL-KDD. The current scope is a binary XGBoost intrusion detection baseline with SHAP-based analysis of false negatives, aimed at understanding missed attacks and feature contributions behind those misses.

Roadmap:

- Extend the NSL-KDD workflow to multiclass classification.
- Add UNSW-NB15 experiments.
- Compare behavior across datasets with cross-dataset evaluation.
